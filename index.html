Etiquetas Extras

Etiqueta pre

La etiqueta pre sirve para respetar los espacios y líneas en blanco, al contrario de la etiqueta párrafo (p)

Hola soy Frida tengo 17 años 

Hola                  soy
        Frida              tengo              17 años

Etiqueta abr

La estiqueta ue se utiliza para las abreviaturas

Según datos de la OMS el consumo excesivo de azúcar provoca daños severos en la salud de los niños

Etiqueta bdo

Si se desea cambiar la dirección de un texto, se utiliza la etiqueta bdo

este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza opara escribir citas textuales

Esto es 100 y 100, TODO O NADA por: Eladio

Etiqueta small

La etiqueta small hace el texto más pequeño

Este texto está dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice, se debe utilizar la etiqueta sub

1443

Etiqueta sup

Si se desea escribir un superíndice, se debe utilizar la etiqueta sup

1669

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time

Mañana es día 2024-11-06

Maña es día 6 de noviembre de 2024

Etiqueta address

Para especificar al navegador que el texto se refiere a una dirección se utiliza la etiqueta address

Loma Dorada No. 100 Colonia Lomas de Rosales CP 89100 Tampico, Tamps.
